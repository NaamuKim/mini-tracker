## tistory 사용자라면?
- [tistory](https://tistory.com)에 로그인 후, [관리]에 들어갑니다.
- [관리]에서 [스킨 편집]을 선택합니다.
- [스킨 편집]에서 [HTML 편집]을 선택합니다.
- [HTML 편집]에서 [포스트]를 선택합니다.
- `</head>` 태그 바로 위에 아래 코드를 붙여넣습니다.
```html
<script src="https://dljue2p0jo5rq.cloudfront.net/mini-tracker-sdk.js"></script>
```
